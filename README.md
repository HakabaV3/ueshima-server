# ueshima-server

## API

- エントリーポイントは`http://ueshima.hakaba.xyz/api/v1`
- `POST /user`以外のリクエストに必ずトークンを付与する
- トークンのキーは`X-Session-Token`

## User

POST /user

### POST /user

#### request

- query

```
?name=kikurage
```

#### response

```
{
    "status": 200,
    "result": {
    	"user": {
    		"id": "dlfajoeij0489u20",
    		"name": "kikurage",
    		"token": "93814h523nrfewjnr0293u"
    		"created": 12345678,
    		"updated": 12345678
    	}
    }
}
```
